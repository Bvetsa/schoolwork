Dialog Practice Activity

Your goal to create an order form that asks the user to input :
1. Their name
2. Street Address
3. City 
4. State
5. Zip code
6. Three separate items that they would like to purchase
7. When you have all the information from the user, output the order 
8. Next, ask the user if the information is correct. They will respond 'y' for yes and 'n' for no. 
9. Ask user for price of each of the 3 items 
10. Output total cost of all 3 items.

 Please submit today and show me when you are done.
